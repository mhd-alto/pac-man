🟡 Pac-Man Game — Beginner Explanation

1. What is this project?

This is a simple Pac-Man game made using HTML, CSS, and JavaScript. You control Pac-Man using your keyboard and try to eat all the dots while avoiding ghosts.

2. How the game is built

The game has 3 main parts:

  • HTML → creates the structure (grid, score)
  • CSS → styles the game (colors, layout)
  • JavaScript → controls the logic (movement, scoring, AI)

3. The Grid (Game Board)

The game board is a grid made of small squares. Each square is a div. 

const width = 28;

This means:

  • The grid has 28 columns
  • Total squares = 28 × 28 = 784

All squares are stored inside an array: 

const squares = [];

Think of this array like a list of all positions in the game.

4. The Layout System

The layout array defines what each square contains: 

0 = pac-dot (food)
1 = wall
2 = ghost area
3 = power pellet
4 = empty space

Each number becomes a visual element using CSS classes.

5. Creating the Board

This function builds the grid: 

for (let i = 0; i < layout.length; i++) {
  const square = document.createElement("div");
  grid.appendChild(square);
  squares.push(square);
}

Step-by-step:

  • Create a new square
  • Add it to the grid
  • Store it in the array
  • Add a class based on layout

6. Pac-Man Movement

Pac-Man moves using arrow keys.

Example: 

if (key === "ArrowLeft") {
  pacmanCurrentIndex -= 1;
}

But before moving, we check:

  • Is there a wall?
  • Is it inside the grid?
  • Is it a ghost area?

This prevents illegal movement.

7. Eating Food

Pac-dots

if (square.classList.contains("pac-dot")) {
  score++;
}

Power pellets

Special food that makes ghosts scared. 

ghost.isScared = true;

After 10 seconds, ghosts return to normal.

8. Ghost System

Ghosts are objects created using a class. 

class Ghost {
  constructor(name, start, speed) {
    this.name = name;
    this.startIndex = start;
    this.speed = speed;
  }
}

Each ghost has:

  • Position
  • Speed
  • Scared state

9. Ghost AI (BFS Algorithm)

Instead of moving randomly, ghosts use BFS (Breadth-First Search).

What is BFS?

BFS is a way to find the shortest path between two points.

How it works:

  1. Start from ghost position
  2. Check all possible moves
  3. Explore step by step
  4. Stop when reaching Pac-Man
const bfs = (start, target) => {
  const queue = [[start]];
  const visited = new Set();

  while (queue.length) {
    const path = queue.shift();
    const current = path[path.length - 1];

    if (current === target) return path;
  }
};

The ghost then follows the first step of this path.

10. Game Rules

Game Over

  • Pac-Man touches a normal ghost

Win

  • All dots are eaten

11. Scoring System

  • Dot = +1
  • Power pellet = +10
  • Ghost = +100
