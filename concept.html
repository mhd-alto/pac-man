JavaScript Concepts in Pac-Man Game

1. Arrow Functions

Shorter way to write functions.

const initializeGame = () => { console.log("Game started!"); }

2. Template Literals

Use backticks and embed variables.

console.log(`Score: ${score}`);

3. forEach()

Loop through arrays easily.

elements.forEach(el => { console.log(el); });

4. setTimeout & setInterval

Run code after delay or repeatedly.

setTimeout(() => {}, 1000); setInterval(() => {}, 500);

5. classList

Modify CSS classes.

el.classList.add("active"); el.classList.remove("hidden");

6. Classes

Create object blueprints.

class Ghost { constructor(name) { this.name = name; } }

7. Arrays

Store multiple values.

const arr = [1,2,3]; arr.push(4);

8. Objects

Store key-value pairs.

const obj = {name: "Pacman"};

9. Switch

Multiple conditions cleaner than if/else.

switch(key) { case "ArrowLeft": break; }

10. Event Listeners

Respond to user input.

document.addEventListener("click", () => {});

11. BFS Algorithm

Find shortest path.

const bfs = () => { // pathfinding }

12. Ternary Operator

Shortcut for if/else.

const result = condition ? "yes" : "no";

13. Spread Operator

Copy arrays.

const newArr = [...arr];

14. Set

Store unique values.

const set = new Set();

15. Audio

Play sound.

const sound = new Audio("sound.mp3"); sound.play();

16. clearInterval

Stop repeating function.

clearInterval(timer);

17. DOM Manipulation

Modify HTML elements.

document.body.innerHTML = "Hello";

18. Modulo Operator

Get remainder.

if (i % 2 === 0) {}

19. Style Manipulation

Change CSS directly.

el.style.color = "red";
