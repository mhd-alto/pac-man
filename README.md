# JavaScript Concepts Summary for Pac-Man Game

This Pac-Man game uses several modern JavaScript concepts that beginners should learn:

## **Core ES6+ Features:**
- **Arrow Functions** - Shorter function syntax `() => {}`
- **Template Literals** - Strings with embedded variables using backticks and `${}`
- **forEach()** - Array looping method
- **Classes** - Blueprints for creating objects (Ghosts)

## **Timing & Events:**
- **setTimeout()** - Run code once after delay
- **setInterval()** - Run code repeatedly at intervals
- **clearInterval()** - Stop repeated code execution
- **Event Listeners** - Respond to keyboard and mouse actions

## **DOM Manipulation:**
- **classList** - Add, remove, or check CSS classes on elements
- **Style manipulation** - Directly change CSS properties
- **Creating/deleting** HTML elements dynamically

## **Data Structures:**
- **Arrays** - Store lists of data
- **Objects** - Store key-value pairs
- **Set** - Store unique values (faster lookup)

## **Algorithms:**
- **BFS (Breadth-First Search)** - Pathfinding for ghost movement
- **Queue** - First-in-first-out data structure

## **Other Key Concepts:**
- **Switch statements** - Clean alternative to multiple if/else
- **Ternary operator** - Shorthand conditional `condition ? true : false`
- **Spread operator (...)** - Expand array elements
- **Audio objects** - Play sound effects and music
- **Modulo operator (%)** - Check grid boundaries

## **Beginner's Learning Path:**
1. Variables (let/const)
2. Functions (regular then arrow)
3. Arrays and objects
4. Conditionals (if/else, switch)
5. Loops (forEach)
6. DOM manipulation
7. Event listeners
8. setTimeout/setInterval
9. Classes
10. Basic algorithms (BFS)

**Key takeaway:** Modern JavaScript emphasizes cleaner syntax (arrow functions, template literals), better organization (classes), and asynchronous programming (timers, events). Start with basics, then gradually incorporate these ES6+ features!
